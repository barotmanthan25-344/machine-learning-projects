# Machine Learning Projects Portfolio

## Overview

This repository contains Machine Learning projects developed using Python and popular data science libraries. The projects cover Supervised Learning, Unsupervised Learning, Classification, Regression, Clustering, and Recommendation Systems.

The purpose of these projects is to demonstrate practical knowledge of:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Data Visualization
- Machine Learning Workflow

## Machine Learning Models Included

### 1. Linear Regression
**Objective:** Predict house prices based on property features.

**Skills Demonstrated:**
- Data preprocessing
- Feature selection
- Regression analysis
- MAE, RMSE, R² evaluation

---

### 2. Polynomial Regression
**Objective:** Capture non-linear relationships between variables.

**Skills Demonstrated:**
- Polynomial feature generation
- Curve fitting
- Model comparison

---

### 3. Logistic Regression
**Objective:** Classification of categorical outcomes.

**Skills Demonstrated:**
- Binary classification
- Probability prediction
- Confusion Matrix

---

### 4. Decision Tree Classifier
**Objective:** Predict outcomes using tree-based learning.

**Skills Demonstrated:**
- Tree visualization
- Feature importance
- Classification analysis

---

### 5. Random Forest
**Objective:** Improve prediction accuracy using ensemble learning.

**Skills Demonstrated:**
- Ensemble methods
- Feature importance
- Overfitting reduction

---

### 6. K-Nearest Neighbors (KNN)
**Objective:** Predict classes based on nearest data points.

**Skills Demonstrated:**
- Distance-based learning
- Feature scaling
- Classification metrics

---

### 7. Support Vector Machine (SVM)
**Objective:** Create optimal decision boundaries for classification.

**Skills Demonstrated:**
- Hyperplane optimization
- Classification
- Accuracy evaluation

---

### 8. K-Means Clustering
**Objective:** Group similar data points without labels.

**Skills Demonstrated:**
- Unsupervised Learning
- Cluster analysis
- Elbow Method

---

### 9. DBSCAN Clustering
**Objective:** Discover clusters based on density.

**Skills Demonstrated:**
- Density-based clustering
- Noise detection
- Cluster visualization
